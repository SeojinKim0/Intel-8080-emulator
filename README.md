# Intel 8080 Emulator

Intel 8080 processor written in Rust. For help message, Run ```cargo r -- -h```

# Running tests

Run ```cargo r -- -f cpudiag.bin -a 0x100```. This will run Microcosm associates 8080/8085 CPU diagnostic test.
